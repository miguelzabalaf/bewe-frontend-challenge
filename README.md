# Bewe: Frontend Challenge (React)

I hope and exceed your expectations. :)

## Run this project

1. Install all dependencies:

```sh
npm install
```

2. Run project:

```sh
npm start
```

If you have any problem, these're my actual global versions:

-   node: v18.15.0
